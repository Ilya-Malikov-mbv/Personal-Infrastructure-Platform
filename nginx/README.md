# Nginx

## Назначение

В данной директории находятся конфигурационные файлы Nginx.

Nginx используется как единая точка входа для всех публичных сервисов инфраструктуры.

Основные задачи:

- обслуживание статического сайта;
- Reverse Proxy;
- HTTPS (Let's Encrypt);
- перенаправление HTTP → HTTPS;
- маршрутизация запросов между сервисами.

---

## Структура

```text
nginx/
└── sites/
    ├── site
    ├── grafana
    └── status
```

---

## Конфигурации

### site

Основной веб-сайт.

```
https://mal-mbv.ru
```

Обслуживает статические файлы проекта.

---

### grafana

Reverse Proxy для Grafana.

```
https://grafana.mal-mbv.ru
```

Перенаправляет запросы на:

```
127.0.0.1:3000
```

---

### status

Reverse Proxy для Uptime Kuma.

```
https://status.mal-mbv.ru
```

Перенаправляет запросы на:

```
127.0.0.1:3001
```

---

## Проверка конфигурации

Перед применением изменений рекомендуется проверить конфигурацию:

```bash
sudo nginx -t
```

Применить изменения:

```bash
sudo systemctl reload nginx
```

## Безопасность

Внутренние сервисы Grafana, Prometheus и Uptime Kuma не доступны напрямую из Интернета.

Все пользовательские запросы проходят через Nginx, который:

- завершает TLS-соединение;
- выполняет маршрутизацию;
- скрывает внутренние порты сервисов;
- обеспечивает единую точку входа в инфраструктуру.
