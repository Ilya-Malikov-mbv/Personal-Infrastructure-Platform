# Diagrams

В данной директории находятся исходные схемы инфраструктуры проекта.

Все диаграммы написаны на языке Mermaid и используются в документации.

## Состав

| Файл                | Описание                                  |
| ------------------- | ----------------------------------------- |
| architecture.mmd    | Общая архитектура инфраструктуры          |
| deployment-flow.mmd | Процесс автоматического развертывания     |
| monitoring.mmd      | Архитектура мониторинга                   |
| request-flow.mmd    | Поток обработки пользовательского запроса |

Диаграммы можно открыть непосредственно на GitHub или использовать в любом редакторе с поддержкой Mermaid.
